Enabling Java in Firefox Ubuntu

Jul 21st, 2015

  1. Download Java jdk-<version>-linux-x64.tar.gz (for 64 bit)
  2. Extract the archive
  3. Run ln -sf /path/to/jdk/jre/lib/amd64/libnpjp2.so ~/.mozilla/plugins/
